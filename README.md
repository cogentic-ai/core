# @cogentic/core

A powerful TypeScript framework for building intelligent AI agents. Built with OpenAI, this framework makes it easy to create, customize, and deploy AI agents for various tasks.

## Features

- 🧠 **Flexible Agent System**: Create custom AI agents with defined capabilities and goals
- 🛠️ **Built-in Tools**: Ready-to-use tools for image analysis and web scraping
- 🔌 **Extensible**: Easy to add new tools and capabilities
- 📝 **Type-Safe**: Full TypeScript support for a robust development experience
- 🚀 **Modern**: Built with Bun and modern JavaScript features

## Installation

```bash
# Using npm
npm install @cogentic/core

# Using yarn
yarn add @cogentic/core

# Using bun
bun add @cogentic/core
```
